# RANSAC算法步骤

  1. 从所有数据中采样n个点，n个点是估计模型的最小点数；
  2. 从采样出的n个点计算出模型；
  3. 通过模型来确定所有数据的内点，当内点率达到指定值，算法终止。否则继续上述过程。

# LO-RANSAC

  1. 1~3同上；
  2. 如果这次计算出的内点比以前某一次计算出的内点数多，那么就执行Local Optimization.
